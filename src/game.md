Описание файла game.h
Файл game.h является общим заголовочным файлом для проекта игры в пинг-понг. Он содержит:

Константы для настройки игры.

Глобальные переменные для хранения состояния игры.

Прототипы функций, которые должны быть реализованы программистами.

Этот файл обеспечивает согласованность данных и функций между всеми частями проекта.

Как использовать game.h
1. Подключение файла
В начале каждого файла (например, draw.c, input.c, update.c) добавьте строку:


#include <game.h>
Это позволит использовать константы, глобальные переменные и прототипы функций, объявленные в game.h.

2. Константы
В файле определены следующие константы:

WIDTH — ширина игрового поля (80 символов).

HEIGHT — высота игрового поля (25 символов).

PADDLE_SIZE — размер ракетки (3 символа).

BALL_SIZE — размер мяча (1 символ).

Используйте эти константы для настройки логики игры.

3. Глобальные переменные
В файле объявлены глобальные переменные, которые хранят состояние игры:

paddle1_y — позиция левой ракетки.

paddle2_y — позиция правой ракетки.

ball_x, ball_y — позиция мяча.

ball_dx, ball_dy — направление движения мяча.

score1, score2 — счет игроков.

Эти переменные объявлены как extern, поэтому их можно использовать в любом файле, но изменять их нужно 
осторожно.

4. Прототипы функций
В файле объявлены прототипы функций, которые должны быть реализованы:

void draw(); — отрисовывает игровое поле.

void input(); — обрабатывает ввод пользователя.

void update(); — обновляет состояние игры.

Каждый программист должен реализовать свою функцию в соответствующем файле.
