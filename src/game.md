# Описание файла `game.h`

Файл **game.h** является общим заголовочным файлом для проекта игры в пинг-понг.  
Он содержит:

- Константы для настройки игры  
- Глобальные переменные для хранения состояния  
- Прототипы функций, реализуемых программистами  

Этот файл обеспечивает согласованность данных и функций между всеми частями проекта.

---

## Как использовать `game.h`

### Подключение файла

В начале каждого файла (`draw.c`, `input.c`, `update.c`) добавьте:

```c
#include "game.h"
```

Это позволит использовать константы, глобальные переменные и прототипы функций, объявленные в `game.h`.

---

## Константы

В файле определены следующие константы:

- `WIDTH` — ширина игрового поля (80 символов)  
- `HEIGHT` — высота игрового поля (25 символов)  
- `PADDLE_SIZE` — размер ракетки (3 символа)  
- `BALL_SIZE` — размер мяча (1 символ)  

Используйте эти константы при реализации логики игры.

---

## Глобальные переменные

Объявлены глобальные переменные, описывающие текущее состояние игры:

- `paddle1_y` — позиция левой ракетки  
- `paddle2_y` — позиция правой ракетки  
- `ball_x`, `ball_y` — позиция мяча  
- `ball_dx`, `ball_dy` — направление движения мяча  
- `score1`, `score2` — счёт игроков  

Все переменные объявлены как `extern`, поэтому доступны во всех файлах проекта.

---

## Прототипы функций

В `game.h` объявлены следующие прототипы:

```c
void draw();   // Отрисовывает игровое поле
void input();  // Обрабатывает ввод пользователя
void update(); // Обновляет состояние игры
```

Каждый программист реализует свою функцию в соответствующем .c файле.
