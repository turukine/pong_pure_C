Описание проекта
Этот проект представляет собой простую игру в пинг-понг, реализованную на языке C. Игра поддерживает двух игроков, управляющих ракетками, и мяч, который отскакивает от ракеток и стен.

Структура проекта
/project_folder/
│
├── game.h      # Общий заголовочный файл с константами и прототипами функций
├── main.c      # Основной файл с функцией main()
├── draw.c      # Файл с функцией отрисовки игрового поля
├── input.c     # Файл с функцией обработки ввода
├── update.c    # Файл с функцией обновления состояния игры
└── README.txt  # Этот файл с инструкциями

Как использовать game.h
Подключение заголовочного файла:
В каждом файле (например, draw.c, input.c, update.c) добавьте в начало строку:

    #include "game.h"

Это обеспечит доступ к общим константам, глобальным переменным и прототипам функций.

Глобальные переменные:
paddle1_y, paddle2_y — позиции ракеток игроков.
ball_x, ball_y — позиция мяча.
ball_dx, ball_dy — направление движения мяча.
score1, score2 — счет игроков.

Эти переменные объявлены как extern в game.h, поэтому их можно использовать в любом файле, но изменять их нужно осторожно.

Константы:
WIDTH — ширина игрового поля (80 символов).
HEIGHT — высота игрового поля (25 символов).
PADDLE_SIZE — размер ракетки (3 символа).
BALL_SIZE — размер мяча (1 символ).

Функции:
void draw();   — отрисовывает игровое поле.
void input();  — обрабатывает ввод пользователя.
void update(); — обновляет состояние игры (движение мяча, счет и т.д.).

Как собрать проект
Убедитесь, что у вас установлен компилятор gcc.
Перейдите в папку проекта.
Выполните команду для компиляции:

    gcc main.c -o game ./game

Запустите игру:

    ./game

Как работать с проектом
Программист 1 (отрисовка):
Реализуйте функцию draw() в файле draw.c.
Используйте глобальные переменные (paddle1_y, paddle2_y, ball_x, ball_y) для отрисовки ракеток и мяча.

Программист 2 (ввод):
Реализуйте функцию input() в файле input.c.
Обрабатывайте нажатия клавиш (a, z, k, m) для управления ракетками.

Программист 3 (логика):
Реализуйте функцию update() в файле update.c.
Обновляйте позицию мяча, проверяйте отскоки и счет.

Интеграция:
После завершения работы над своими частями, объедините код и протестируйте игру.

Правила игры
Игрок 1 управляет левой ракеткой с помощью клавиш a (вверх) и z (вниз).
Игрок 2 управляет правой ракеткой с помощью клавиш k (вверх) и m (вниз).
Мяч отскакивает от ракеток и стен.
Если мяч выходит за пределы поля, противнику засчитывается очко.
Игра заканчивается, когда один из игроков набирает 21 очко.
